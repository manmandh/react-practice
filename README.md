- Typescript

## HOW TO RUN

- Dillinger requires Node.js v18+ to run.
- Install the dependencies and devDependencies and start the server.

```sh
cd react-practice
rm -rf package-lock.json
npm i
npm run dev

Home
http://localhost:3000

```
new terminal
